# AI Flavor Detector

检测文本的 AI 生成特征，分析句式、词汇、语气等维度，帮助识别和去除文本中的"AI 味"。

## 功能特点

- **6 维检测**：句式结构、词汇特征、主语语气、具体性、网络用语、句首模式
- **朱雀专项检测**：针对朱雀 AI 检测器的专项分析
- **逐段标注**：精确定位问题位置和关键词
- **改写建议**：提供具体的去 AI 化改写示例

## 安装

将本目录放入 Claude Code 的技能目录：

```bash
# macOS/Linux
cp -r ai-flavor-detector ~/.claude/skills/

# 或使用符号链接
ln -s /path/to/ai-flavor-detector ~/.claude/skills/
```

## 使用方法

在 Claude Code 中触发以下指令：

```
检查 AI 味
检测 AI 味
分析 AI 味
这篇文章 AI 味重吗
帮我看下这段文字像 AI 写的吗
去 AI 化检查
```

### 示例

```
用户：检查 AI 味 [粘贴文本]

AI:
AI 味评分：7/10

问题分析：
1. [高优先级] 第 2 段："值得注意的是" - 高频连接词
2. [中优先级] 全文缺少具体时间细节
3. [低优先级] 段落长度过于均匀

修改建议：
...
```

## 检测维度

| 维度 | 检测内容 |
|------|----------|
| 句式结构 | 三段式法则、过度结构化、段落长度、句子复杂度 |
| 词汇特征 | 连接词频率、量化词汇、AI 特有词汇、禁用词 |
| 主语语气 | 主语使用、语气判断、祈使句、疑问句 |
| 具体性 | 时间细节、地点细节、个人经历、数据精确度 |
| 网络用语 | 网络流行语、使用自然度、口语化表达 |
| 句首模式 | 句首重复、被动语态、名词化 |

## 朱雀 AI 检测

针对朱雀 AI 检测器的专项检测包括：

- **连接词密度分析**：结构连接词、转折连接词、补充连接词
- **结构化程度**：序号列表、三段式结构
- **括号使用**：圆括号、方括号及说明性标记
- **破折号使用**：频繁补充说明
- **段落结尾标点**：段落规整度分析

## 反朱雀策略

1. **思维过程真实化**：展现真实思考过程
2. **表达不完美化**：段落长短不一、口语化表达
3. **情感波动自然化**：加入情感和看法
4. **视角切换自然化**：我/我们/你的人称切换
