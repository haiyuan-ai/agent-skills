# 朱雀 AI 检测详细指南

基于"反朱雀 AI 检测提示词（姚金刚 V2.0）"的完整检测标准。

---

## 朱雀检测核心原理

朱雀检测通过分析文本的结构化程度、连接词密度、标点使用模式等特征来判断是否为 AI 生成。

---

## 一、连接词密度检测

### 高优先级检测的连接词

| 类别 | 连接词 | 检测规则 |
|------|--------|----------|
| **结构连接词** | 首先、其次、再次、然后、此外、总之、总而言之 | 每 3-4 句出现 1 个即判定为 AI |
| **转折连接词** | 然而、但是、其实、实际上 | 频率过高判定为 AI |
| **补充连接词** | 还有啊、其实啊、值得注意的是 | 明显 AI 特征 |
| **时态连接词** | 随着、发展、在...时代、在...今天 | 开篇使用判定为 AI |
| **总结连接词** | 综上所述、由此可见、可以说 | 结尾使用判定为 AI |

### 检测示例

**AI 特征文本**：
```
随着人工智能技术的发展，各行各业都在经历变革。
首先，医疗领域引入了 AI 诊断系统。
其次，教育行业也开始使用智能教学。
此外，金融行业的风控也离不开 AI。
总之，AI 正在改变我们的生活。
```

**人味文本**：
```
AI 这玩意儿现在真是无处不在了。
医院里用它看片子，学校用它讲课，连银行风控都靠它。
说真的，有时候都觉得离了 AI 日子都没法过了。
```

## 二、序号和列表检测

### 禁用特征

- 中文序号：一、二、三、四...
- 阿拉伯数字：1. 2. 3. 4....
- 字母序号：A. B. C. D....
- 符号列表：- * •

### 检测规则

真人写观点时很少用序号列举，而是用自然语言过渡。

**AI 特征**：
```
选择产品时要注意以下几点：
一、品牌知名度
二、产品质量
三、售后服务
```

**人味写法**：
```
选产品这事儿，我一般先看牌子，大厂的东西起码有保障。
然后看看质量怎么样，用料足不足。
最后还得瞅瞅售后，坏了能找到人修。
```

---

## 三、括号使用说明检测

### 禁用特征

- 圆括号：（说明内容）
- 方括号：[补充信息]
- 六角括号：〔注释〕

### 检测规则

真人很少用括号做补充说明，而是直接融入正文。

**AI 特征**：
```
这款产品（尤其是粉色版本）非常受欢迎，销量（截至上个月）已经突破百万。
```

**人味写法**：
```
这款产品粉色的那个版本卖得特别好，到上个月销量已经破百万了。
```

---

## 四、破折号使用检测

### 检测特征

频繁使用破折号进行补充说明：

```
这个功能——正如我们之前提到的——代表了技术进步。
结果——出乎所有人意料——非常好。
```

### 检测规则

自然写作中破折号使用很少，AI 则偏爱使用。

---

## 五、段落结尾标点检测

### 检测模式

| 写作类型 | 段落结尾特征 |
|----------|--------------|
| 真人写作 | 段落结尾常不使用标点 |
| AI 写作 | 每个段落都有句号结尾 |

### 检测规则

观察段落是否都以标点结束，全部有标点的判定为 AI。

---

## 六、段落规整度检测

### AI 特征

- 段落长度均匀（每段 80-120 字）
- 每段都有主题句和总结句
- 段落之间过渡平稳

### 人味特征

- 段落长短不一
- 有超长段落（300+ 字）
- 有超短段落（一句话成段）
- 段落之间跳跃较大

---

## 七、反朱雀核心策略

### 1. 思维过程真实化

展现真实的思考过程：

- "对了，还有个事情想说..."
- "写到这突然想起来..."
- "其实我之前也纠结过这个问题"
- 在某处提出问题后自己回答

### 2. 表达不完美化

- 段落长短不一
- 口语化表达和不太规范的句子
- 某些观点可能会多说几句，而不是面面俱到
- 用自然的方式过渡，而不是明显的结构词

### 3. 情感波动自然化

- 加入自己的情感和看法，有时甚至有点主观
- 和读者互动，问问题，引发思考
- 该感叹时就感叹，但不要太夸张
- 整体看起来随性自然

### 4. 视角切换自然化

| 场景 | 使用人称 |
|------|----------|
| 讲述个人经历 | "我" |
| 分享共同经验 | "我们" |
| 直接对读者建议 | "你" |
| 讨论普遍现象 | 第三人称 |

## 八、朱雀检测报告模板

### 输出格式

```
## 朱雀专项检测

### 连接词密度：高/中/低
检测到的连接词：
- "首先"（第 2 段）
- "其次"（第 3 段）
- "总之"（最后一段）

### 结构化程度：高/中/低
- 发现序号列表：3 处
- 段落长度均匀度：过高

### 括号使用：X 处
- （尤其是粉色版本）- 第 1 段
- （截至上个月）- 第 2 段

### 破折号使用：X 处
- 第 3 段使用 2 处破折号

### 段落结尾标点：全部使用/部分使用/基本不用

### 人味信号：强/中/弱
- 个人经历：无
- 情感表达：较少
- 口语化：较少
```

## 九、改写建议模板

### 针对连接词过多
```
原文：首先，我们需要了解产品的基本功能。其次，要考虑价格因素。最后，还要看售后服务。
改写：选产品嘛，得先看看基本功能怎么样，价格能不能接受，售后也得考虑考虑。
```

### 针对序号列表
```
原文：
选择产品要注意：
一、品牌
二、质量
三、价格
改写：选产品这事儿，牌子、质量、价格都得看，一个都不能少。
```

### 针对括号说明
```
原文：这款产品（尤其是新版本）非常受欢迎（销量已破百万）。
改写：这款产品的新版本卖得特别好，销量已经破百万了。
```

## 十、评分标准

| 检测项 | AI 特征 | 人味特征 |
|--------|---------|----------|
| 连接词密度 | >5 处/300 字 | <2 处/300 字 |
| 序号使用 | 有 | 无 |
| 括号使用 | >3 处 | 0-1 处 |
| 破折号使用 | >2 处 | 0-1 处 |
| 段落规整度 | 过于均匀 | 长短不一 |
| 段落结尾标点 | 全部使用 | 部分或不用 |

**综合判定**：
- 4 项以上 AI 特征：AI 味明显（7-10 分）
- 2-4 项 AI 特征：有 AI 痕迹（4-6 分）
- 0-1 项 AI 特征：人味很浓（0-3 分）
